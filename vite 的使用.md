# vite 的使用

## 安装

```bash
yarn global add create-vite-app@1.18.0
```

## 创建项目

 ```bash
create-vite-app wheel-ui-vue3.0
cva wheel-ui-vue3.0
 ```

## 运行项目

```bash
  cd wheel-ui-vue3.0
  npm install (or `yarn`)
  npm run dev (or `yarn dev`)
```

## vue2和vue3的区别

1. vue3 template 支持多个根标签，vue2 不支持
2. vue3 是createApp(组件), vue2是new Vue({template, render})
